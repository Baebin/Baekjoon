## **The Four Fundamental Arithmetic Operation**

| Time Limit | Memory Limit |
| --- | --- |
| 1 Second | 256 MB |

___

## Problem
Print
```
A+B
A-B
A*B
A/B(Quotient)
A%B
```

___

## Input
A B (1 ≤ A, B ≤ 10,000)

___

## Output
```
A+B
A-B
A*B
A/B(Quotient)
A%B
```

___

| Example Input | Example Output |
| --- | --- |
| 7 3 | 10</br>4</br>21</br>2</br>1 |
