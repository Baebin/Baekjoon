## **Print up to N (Reverse)**

| Time Limit | Memory Limit |
| --- | --- |
| 1 Second | 128 MB |

___

## Problem
```
Print the numbers (N ~ 1).
```

___

## Input (1 ≤ N ≤ 100,000)
N

___

## Output
```
1
2
3
...
N
```

___

| Example Input | Example Output |
| --- | --- |
| 5 | 1</br>2</br>3</br>4</br>5 |
