## **Compare Two Numbers**

| Time Limit | Memory Limit |
| --- | --- |
| 1 Second | 128 MB |

___

## Problem
```
Compare two numbers.
```

___

## Input (-10,000 ≤ A, B ≤ 10,000)
A B

___

## Output
| Condition | Output |
| --- | --- |
| (a<b) | < |
| (a=b) | == |
| (a>b) | > |

___

| Example Input | Example Output |
| --- | --- |
| 1 2 | < |
| 10 2 | > |
| 5 5 | == |
