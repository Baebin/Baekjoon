## **Number less than X**

| Time Limit | Memory Limit |
| --- | --- |
| 1 Second | 256 MB |

___

## Problem
```
Print the number less than X.
```

___

## Input (1 ≤ N, X ≤ 10,000)
```
N X
A1 A2 A3 A4
...
```

___

| Example Input | Example Output |
| --- | --- |
| 10 5</br>1 10 4 9 2 3 8 5 7 6 | 1 4 2 3 |
