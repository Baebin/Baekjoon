## **Print up to N (Beauty)**

| Time Limit | Memory Limit |
| --- | --- |
| 1 Second | 128 MB |

___

## Problem
```
Print the numbers (1 ~ N).
```

___

## Input (1 ≤ N ≤ 100,000)
N
A B
A B
A B
...

___

## Output
```
1
2
3
...
N
```

___

| Example Input | Example Output |
| --- | --- |
| 5 | |
| 1 1 | Case #1: 2 |
| 2 3 | Case #2: 5 |
| 3 4 | Case #3: 7 |
| 9 8 | Case #4: 17 |
| 5 2 | Case #5: 7 |
