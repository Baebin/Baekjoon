## **The Four Fundamental Arithmetic Operation**

| Time Limit | Memory Limit |
| --- | --- |
| 1 Second | 256 MB |

___

## Problem
Print
```
(A+B)%C
((A%C) + (B%C))%C
(A×B)%C
((A%C) × (B%C))%C
```

___

## Input
A B C (2 ≤ A, B, C ≤ 10000)
___

## Output
```
(A+B)%C
((A%C) + (B%C))%C
(A×B)%C
((A%C) × (B%C))%C
```

___

| Example Input | Example Output |
| --- | --- |
| 5 8 4 | 1</br>1</br>0</br>0 |
